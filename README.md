# SpendSense User Manual

## Overview
SpendSense is a personal finance management app that helps you track expenses, manage budgets using the 50/30/20 rule, and monitor your spending habits.

---

## Getting Started

### 1. Registration
- Open the app and tap "Register"
- Enter your Name, Email (valid format required), and Password (minimum 6 characters)
- Tap "Register" to create your account

### 2. PIN Setup
- Create a 4-digit PIN for security
- Confirm your PIN by entering it again
- This PIN protects your account on login

### 3. Budget Setup
- **Step 1:** Enter your total budget amount
- **Step 2:** Select your budget period (Weekly, Bi-weekly, or Monthly)
- **Step 3:** Allocate your budget using the 50/30/20 rule:
  - **Needs (50%):** Essential expenses (bills, groceries, rent)
  - **Savings (20%):** Money set aside for savings
  - **Wants (30%):** Non-essential spending (entertainment, dining out)
- Percentages update automatically as you adjust amounts

---

## Main Features

### Dashboard
- View your current budget status with a progress bar
- See remaining balance and spending percentage
- Quick access to recent transactions via "See All"
- Navigation bar at bottom for quick access to all features

### Adding Transactions
- Tap the "+" button on the dashboard
- Select transaction type: **Expense** or **Cash In**
- Enter amount and select a category
- Add optional notes
- **Note:** You must set up a budget before adding expenses

### Records
- View all your transaction history
- Filter by: All, Expenses only, or Income only
- Search transactions by category or notes
- Delete transactions by tapping the trash icon

### Analytics (Track)
- View spending breakdown by time period: Week, Month, or All
- See total spent vs total income
- Transaction count summary
- **Insight Card:** Shows your top spending category
- **Budget Breakdown:** Visual progress bars for Needs, Savings, Wants
- **Category Breakdown:** Horizontal bar chart showing spending per category with percentages

- On the login screen, tap "Forgot Password?"
- Enter your registered email address

## Tips for Best Use
1. Set up your budget immediately after registration
2. Log transactions as they happen for accurate tracking
3. Check Analytics weekly to understand spending patterns
4. Use the 50/30/20 rule as a guideline, adjust percentages to fit your needs
5. Reset budget at the start of each budget period for fresh tracking

## Technical Notes
- Single-user app (one account per device)
- Data stored locally on device
- PIN required for login security
- Budget progress bar: Green (<50%), Orange (50-80%), Red (>80%)
